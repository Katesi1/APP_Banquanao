# APP BÁN QUẦN ÁO BASIC

## 1.Clone Code:

Mở terminal và gõ lệnh "git clone https://github.com/Katesi1/APP_Banquanao.git"

## 2.Bật Xampp:

* Start Module Apache,MySQL để có thể vào trang cơ sở dữ liệu theo đường dẫn: http://localhost/phpmyadmin/

![image](https://github.com/user-attachments/assets/60658543-3e73-445c-869e-e1f35168094b)

* Tại trang phpmyadmin: Ấn vào "Mới" bên thanh bên trái để tạo cơ sở dữ liệu -> Đặt tên csdl cho csdl

![image](https://github.com/user-attachments/assets/97d0cb86-a107-43bb-8ca4-b2dc242f1267)

* Tại cơ sở dữ liệu vừa tạo -> trỏ vào Nhập của thanh trên -> Chọn tệp quanao.sql -> Nhập

![image](https://github.com/user-attachments/assets/3489b171-c5a9-495c-99c8-f5e15cd95c78)

* Khi nhập thành công sẽ có cấu trúc gồm 5 bảng dữ liệu

![image](https://github.com/user-attachments/assets/b08ced06-580a-452d-bc48-24268a882eec)
